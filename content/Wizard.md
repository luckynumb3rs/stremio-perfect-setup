[⚡ DEBRID / 🧲 P2P / 🌐 HTTP]

Many of you are already aware of my comprehensive guide and have been using it a lot (I know because of the many support questions in my DMs 😉), so thank you for the love ❤️, I'm glad you're enjoying it! 

>BTW, the guide has it's own domain now: [**numb3rs.stream**](https://numb3rs.stream/) 🎉.

Because of this popularity, I wanted to take it a step further and make it even easier for all of you to set up your accounts (and take some load off my inbox 😊). 

Hence, I just added a new [**🔮 Wizard**](https://numb3rs.stream/wizard/) to the guide, which should make the whole setup much easier for anyone who does not want to manually go through every addon, import screen, manifest URL, and configuration page. The full guide is still the main reference if you want to understand everything properly, but the wizard now gives you a much faster way to get a ready-to-use setup in just a few clicks.

The wizard sets everything up automatically, with a bit of input from you:

* **🔑 Account**: Sign in to your account or create a new one directly.
   * **Profiles** are of course also supported, you can even create new profiles for existing accounts.
* **🎥 TMDB/TVDB**: Add your own keys, or skip to use shared ones directly.
   * **⚠️ NOTE**: Using own keys is still highly recommended to avoid shared key limits or possible issues.
* **⚡ Debrid / 🧲 P2P / 🌐 HTTP**: Choose the streaming approach you want to use.
* **📚 AIOStreams**: Configure your preferences, such as language, subtitles, formatting, and other main options directly from the wizard.
* **🔎 AIOMetadata**: Select the catalog categories you want to have enabled directly from the wizard.
* **💎 Collections**: The matching collections configuration (and some other Nuvio settings) are also automatically configured, so the Nuvio account is ready out-of-the-box.
* **🎯 Trakt**: You can integrate Trakt with *AIOMetadata* directly from the wizard.

So, instead of manually configuring each part separately, the wizard configures everything, installs the addons for you, and gives you a **ready-to-use account**. Once it's done, you'll get the credentials used for the addons, which you can download and store it for safekeeping, in case you need to customize them further.

The main Nuvio guide post is still [**here**](https://www.reddit.com/r/Nuvio/comments/1t9yub7/nuvio_perfect_setup_beginners_guide/), and the repo is [here](https://github.com/luckynumb3rs/stremio-perfect-setup) for anyone who wants to check the files and assets directly.

That's it, I hope you like it, and enjoy!
